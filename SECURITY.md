# Security Policy

## Supported Versions

Currently, only the latest major release receives active security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Security is a core pillar of the Enjay Smart HelpDesk architecture (especially concerning RBAC and JWT handling). If you discover a security vulnerability within this project, please **do not** disclose it publicly.

Instead, please send an email to **security@enjay.com**. 
Include:
- A detailed description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact (e.g., Data exfiltration, privilege escalation).

We will acknowledge receipt of your vulnerability report within 48 hours and strive to issue a patch within 7 days.
